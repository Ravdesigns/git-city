# git-city — Product context

## Register
brand. git-city is an expressive, interactive artifact where the design is the
product. It is a portfolio piece, not an app serving a task. Apply product-grade
rigor to the functional bits (the repo form, buttons, the game HUD), but the
whole reads as a nocturnal, cinematic experience.

## Product purpose
Turn any GitHub repo's commit history into an explorable 3D night-city: one tower
per file changed in a commit, height = lines touched, colour = file type, the
timeline running oldest to newest. Visitors can build a city from any repo (live,
in the browser) and fly a glowing orb down its longest avenue as a game. No
backend, no build step: one HTML file plus vendored three.js.

## Users
- Developers and designers curious what a repo "looks like".
- Portfolio visitors judging Ravindra's craft in ~10 seconds.
- People who land, poke, and should immediately get what they are seeing.

## Brand voice
Three words: cinematic, mechanical, playful. Nocturnal and atmospheric (a city
at night seen from above), precise and technical (monospace, keyboard-driven),
with a light, confident wink ("you reached HEAD", "match. munch. maybe more."
energy). Lowercase, terse, technical-poetic. Never corporate, never cute.

## Tone of copy
Lowercase. Short. Says the true thing plainly ("turn any repo into a skyline",
"one tower = a file in a commit"). Labels are UPPERCASE with wide tracking.
No em dashes. No restated headings.

## Anti-references (do not look like these)
Generic SaaS dashboards. Neon-crypto-on-black. Glassmorphism-as-decoration.
The hero-metric template. AI-slop landing pages with centered icon-title cards.
Light/clinical data-viz. If it could be mistaken for a Vercel-template clone,
it has failed.

## Strategic principles
1. The data IS the city. The mapping must stay legible (legend + hover inspect).
2. Atmosphere over chrome. UI sits quietly at the edges; the scene is the hero.
3. Offline-first, no backend, no secrets. Everything runs client-side.
4. Identity is Zop.Dev DNA: dark, monospace, a single hot orange signature.
5. Reward exploration. There is always one more thing to poke (palette, fly).
